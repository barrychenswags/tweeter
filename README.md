## Dependencies
- Mongodb
- Express
- Chance
- body-parser

## Getting Started
- Clone the repo
- Install all dependencies using the `npm install` command.
- Run 'mongo' and set up a database.
- Run the server using the `npm start` command.
- Go to <http://localhost:8080/> in your browser.
